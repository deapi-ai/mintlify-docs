---
title: "Video Generation"
openapi: "/openapi-v2.json POST /api/v2/videos/generations"
---

Generate a video from a text prompt. Returns a `request_id` for status polling.

<Note>
**Prerequisite:** Consult the [Model Selection](/api/v2/utilities/models) endpoint to identify a valid model `slug` and check specific **limits** (resolution, duration, framerate).
</Note>
