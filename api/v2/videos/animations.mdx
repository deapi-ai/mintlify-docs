---
title: "Video Animation (Image-to-Video)"
openapi: "/openapi-v2.json POST /api/v2/videos/animations"
---

Animate a still image into a video using motion synthesis. Returns a `request_id` for status polling.

<Note>
**Prerequisite:** Consult the [Model Selection](/api/v2/utilities/models) endpoint to identify a valid model `slug` and check specific **limits**.
</Note>
