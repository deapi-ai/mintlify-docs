---
title: "Video Upscale"
openapi: "/openapi-v2.json POST /api/v2/videos/upscales"
---

Increase the resolution of a video using AI super-resolution. Returns a `request_id` for status polling.

<Note>
**Prerequisite:** Consult the [Model Selection](/api/v2/utilities/models) endpoint to identify a valid model `slug` and check specific **limits** (including supported `scale` range, maximum input resolution, and duration).
</Note>
