---
title: "Video Audio Sync (Audio-to-Video)"
openapi: "/openapi-v2.json POST /api/v2/videos/audio-syncs"
---

Generate a video synchronized with an audio track plus an optional text prompt. Returns a `request_id` for status polling.

<Note>
**Prerequisite:** Consult the [Model Selection](/api/v2/utilities/models) endpoint to identify a valid model `slug` and check specific **limits**.
</Note>
