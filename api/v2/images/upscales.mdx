---
title: "Image Upscale"
openapi: "/openapi-v2.json POST /api/v2/images/upscales"
---

Increase the resolution of an image using AI super-resolution. Returns a `request_id` for status polling.

<Note>
**Prerequisite:** Consult the [Model Selection](/api/v2/utilities/models) endpoint to identify a valid model `slug` and check specific **limits** (including supported `scale` range and maximum input resolution).
</Note>
