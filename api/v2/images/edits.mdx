---
title: "Image Edit"
openapi: "/openapi-v2.json POST /api/v2/images/edits"
---

Image Edit transforms an existing image based on a text prompt (image-to-image). Returns a `request_id` for status polling.

<Note>
**Prerequisite:** Consult the [Model Selection](/api/v2/utilities/models) endpoint to identify a valid model `slug` and check specific **limits**.
</Note>
