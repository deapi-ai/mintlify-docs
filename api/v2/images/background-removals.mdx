---
title: "Image Background Removal"
openapi: "/openapi-v2.json POST /api/v2/images/background-removals"
---

Remove the background from an image, producing a transparent PNG output. Returns a `request_id` for status polling.

<Note>
**Prerequisite:** Consult the [Model Selection](/api/v2/utilities/models) endpoint to identify a valid model `slug`.
</Note>
