---
title: "Text-to-Speech (TTS)"
api: "POST https://api.deapi.ai/api/v1/client/txt2audio"
---

Convert text to speech audio.

**Prerequisite:** Consult the [Model Selection](/api/model-selection) endpoint to identify a valid model `slug`.

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField body="model" type="string" required>
  Model slug from Model Selection endpoint.
</ParamField>

<ParamField body="text" type="string" required>
  Text to convert to speech.
</ParamField>

<ParamField body="voice" type="string">
  Voice ID to use.
</ParamField>

<ParamField body="speed" type="number" default="1.0">
  Playback speed multiplier. Fast (2.0) = 0.5× cost, Slow (0.5) = 2× cost.
</ParamField>

---

## Calculate Price

**Endpoint:** `POST https://api.deapi.ai/api/v1/client/txt2audio/price-calculation`

```bash
curl --request POST \
  --url https://api.deapi.ai/api/v1/client/txt2audio/price-calculation \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "tts-model",
    "text": "Hello world",
    "speed": 1.0
  }'
```
