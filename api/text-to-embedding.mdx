---
title: "Text-to-Embedding"
api: "POST https://api.deapi.ai/api/v1/client/txt2embedding"
---

Generate vector embeddings from text for semantic search and RAG.

**Prerequisite:** Consult the [Model Selection](/api/model-selection) endpoint to identify a valid model `slug`.

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField body="model" type="string" required>
  Model slug from Model Selection endpoint.
</ParamField>

<ParamField body="text" type="string" required>
  Text to generate embeddings for.
</ParamField>

---

## Calculate Price

**Endpoint:** `POST https://api.deapi.ai/api/v1/client/txt2embedding/price-calculation`

```bash
curl --request POST \
  --url https://api.deapi.ai/api/v1/client/txt2embedding/price-calculation \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "embedding-model",
    "text": "Sample text for embedding"
  }'
```
