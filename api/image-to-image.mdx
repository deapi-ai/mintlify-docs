---
title: "Image-to-Image"
api: "POST https://api.deapi.ai/api/v1/client/img2img"
---

Transform existing images based on prompts.

**Prerequisite:** Consult the [Model Selection](/api/model-selection) endpoint to identify a valid model `slug`.

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField body="model" type="string" required>
  Model slug from Model Selection endpoint.
</ParamField>

<ParamField body="prompt" type="string" required>
  Text description for transformation.
</ParamField>

<ParamField body="image" type="string" required>
  Base64 encoded image or URL.
</ParamField>

<ParamField body="strength" type="number" default="0.75">
  Transformation strength (0-1).
</ParamField>

<ParamField body="steps" type="integer" default="20">
  Number of inference steps.
</ParamField>

---

## Calculate Price

**Endpoint:** `POST https://api.deapi.ai/api/v1/client/img2img/price-calculation`

```bash
curl --request POST \
  --url https://api.deapi.ai/api/v1/client/img2img/price-calculation \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "img2img-model",
    "steps": 20
  }'
```
