---
title: "Text-to-Image"
api: "POST https://api.deapi.ai/api/v1/client/txt2img"
---

**Prerequisite:** Consult the [Model Selection](/api/model-selection) endpoint to identify a valid model `slug`, check specific **limits** and **features**, and verify **LoRA** availability.

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField body="model" type="string" required>
  Model slug from Model Selection endpoint.
</ParamField>

<ParamField body="prompt" type="string" required>
  Text description of the image to generate.
</ParamField>

<ParamField body="width" type="integer" default="512">
  Image width in pixels.
</ParamField>

<ParamField body="height" type="integer" default="512">
  Image height in pixels.
</ParamField>

<ParamField body="steps" type="integer" default="4">
  Number of inference steps.
</ParamField>

<ParamField body="seed" type="integer">
  Random seed for reproducibility.
</ParamField>

<ParamField body="lora" type="string">
  Optional LoRA model identifier.
</ParamField>

---

## Calculate Price

**Endpoint:** `POST https://api.deapi.ai/api/v1/client/txt2img/price-calculation`

Calculate the price before running the job.

```bash
curl --request POST \
  --url https://api.deapi.ai/api/v1/client/txt2img/price-calculation \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "flux-schnell",
    "width": 512,
    "height": 512,
    "steps": 4
  }'
```
