---
title: "Video-to-Text (Transcription)"
api: "POST https://api.deapi.ai/api/v1/client/vid2txt"
---

Transcribe video from URL (e.g., YouTube).

**Prerequisite:** Consult the [Model Selection](/api/model-selection) endpoint to identify a valid model `slug`.

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField body="model" type="string" required>
  Model slug from Model Selection endpoint.
</ParamField>

<ParamField body="url" type="string" required>
  Video URL to transcribe.
</ParamField>

<ParamField body="include_ts" type="boolean" default="false">
  Include timestamps in transcription.
</ParamField>

---

## Calculate Price

**Endpoint:** `POST https://api.deapi.ai/api/v1/client/vid2txt/price-calculation`

```bash
curl --request POST \
  --url https://api.deapi.ai/api/v1/client/vid2txt/price-calculation \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "transcription-model",
    "url": "https://youtube.com/watch?v=example"
  }'
```
