---
title: "Image Background Removal"
api: "POST https://api.deapi.ai/api/v1/client/img-rmbg"
---

Remove background from images.

**Prerequisite:** Consult the [Model Selection](/api/model-selection) endpoint to identify a valid model `slug`.

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField body="model" type="string" required>
  Model slug from Model Selection endpoint.
</ParamField>

<ParamField body="image" type="string" required>
  Base64 encoded image or URL.
</ParamField>

---

## Calculate Price

**Endpoint:** `POST https://api.deapi.ai/api/v1/client/img-rmbg/price-calculation`

```bash
curl --request POST \
  --url https://api.deapi.ai/api/v1/client/img-rmbg/price-calculation \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "rmbg-model",
    "image": "https://example.com/image.jpg"
  }'
```
