---
title: "Image-to-Text (OCR)"
api: "POST https://api.deapi.ai/api/v1/client/img2txt"
---

Extract text from images using OCR/VLM.

**Prerequisite:** Consult the [Model Selection](/api/model-selection) endpoint to identify a valid model `slug`.

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer YOUR_API_KEY`
</ParamField>

<ParamField body="model" type="string" required>
  Model slug from Model Selection endpoint.
</ParamField>

<ParamField body="image" type="string" required>
  Base64 encoded image or URL.
</ParamField>

<ParamField body="prompt" type="string">
  Optional prompt for context.
</ParamField>

---

## Calculate Price

**Endpoint:** `POST https://api.deapi.ai/api/v1/client/img2txt/price-calculation`

```bash
curl --request POST \
  --url https://api.deapi.ai/api/v1/client/img2txt/price-calculation \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "ocr-model",
    "image": "https://example.com/image.jpg"
  }'
```
