<Info>
**Consider using webhooks or WebSockets instead of polling.**

Polling requires repeated API calls and adds latency. For production applications, we recommend:
- **[Webhooks](/execution-modes-and-integrations/webhooks)** — Results pushed to your server automatically (best for backends)
- **[WebSockets](/execution-modes-and-integrations/websockets)** — Real-time updates with live previews (best for interactive UIs)

See [Execution Modes](/execution-modes-and-integrations/execution-modes-and-http-queue) for a full comparison.
</Info>
