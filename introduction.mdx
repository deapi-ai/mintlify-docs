**deAPI** is a low-cost, reliable, and high-performance cloud infrastructure for deploying and scaling AI models. We provide developers with fast, production-ready inference capabilities — from language models to image, video, and speech generation — so they can focus on building powerful AI-driven applications without worrying about infrastructure.

DeAPI leverages the distributed power of thousands of GPUs across the Gamerhash AI network, making it cheap and reliable.

**API Reference for Your AI Assistant**

If you are building an AI Agent or integrating AI assistance in your development workflow, use the **llms.txt** file below to provide your model with structured access to the complete DeAPI reference.
Simply include or link to this file — it contains all core endpoints, canonical links, and usage documentation for the DeAPI platform.

```
https://deapi.ai/llms.txt
```
