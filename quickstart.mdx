**deAPI** is a high-performance inference platform built specifically for developers seeking a fast, reliable, and simple way to integrate cutting-edge AI capabilities into their applications. This guide will help you quickly set up your account, obtain your API key, load credits, and begin using our API.

---

### 1. Sign In to Your Account

Create your free account at [deapi.ai](https://app.deapi.ai/register) to get started. All new users receive a **\$5 sign-up bonus** with a **Basic** account ‚Äî no credit card required.

---

### 2. Obtain Your API Key

deAPI uses token-based authentication. Every request must include your personal API key, provided via an HTTP header:

```
Authorization: Bearer YOUR_API_KEY
```

To get your API key:

- Sign up or log in to your account.
- Navigate to Dashboard ‚Üí Settings ‚Üí API Keys.
- Click "Create new secret key".
- Your API key will be generated instantly.

---

| **Basic**    | \$5 welcome bonus, conservative rate limits for testing |

### 4. Start Using the API

Once your API key is activated and your account has sufficient credits, you can start making requests to deAPI immediately.

**Available Services (v2):**

- **Image Generation** ‚Äî `POST /api/v2/images/generations` ‚Äî Generate images from text prompts using AI diffusion models.
- **Image Edit** ‚Äî `POST /api/v2/images/edits` ‚Äî Transform existing images with text prompts.
- **Image OCR** ‚Äî `POST /api/v2/images/ocr` ‚Äî Extract text from images.
- **Image Background Removal** ‚Äî `POST /api/v2/images/background-removals` ‚Äî Transparent PNG output.
- **Image Upscale** ‚Äî `POST /api/v2/images/upscales` ‚Äî AI super-resolution for images.
- **Video Generation** ‚Äî `POST /api/v2/videos/generations` ‚Äî Generate videos from text prompts.
- **Video Animation (Image-to-Video)** ‚Äî `POST /api/v2/videos/animations` ‚Äî Animate still images.
- **Video Audio Sync** ‚Äî `POST /api/v2/videos/audio-syncs` ‚Äî Generate video from audio + prompt.
- **Video Replacement** ‚Äî `POST /api/v2/videos/replacements` ‚Äî Swap characters in videos.
- **Video Background Removal** ‚Äî `POST /api/v2/videos/background-removals` ‚Äî Keyed/transparent video (v2 only).
- **Video Upscale** ‚Äî `POST /api/v2/videos/upscales` ‚Äî AI super-resolution for videos.
- **Text-to-Speech** ‚Äî `POST /api/v2/audio/speech` ‚Äî Preset voices, voice cloning, voice design.
- **Text-to-Music** ‚Äî `POST /api/v2/audio/music` ‚Äî Tracks with lyrics, tempo, key, style control.
- **Transcription** ‚Äî `POST /api/v2/audio/transcriptions` ‚Äî Unified audio + video (YouTube, X, Twitch, Kick, TikTok, X/Twitter Spaces, file uploads).
- **Text-to-Embedding** ‚Äî `POST /api/v2/embeddings` ‚Äî Vector embeddings for semantic search and RAG.
- **Prompt Enhancement** ‚Äî `/api/v1/client/prompt/*` ‚Äî AI-powered prompt boosters (shared with v1).
